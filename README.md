🛒 E-Commerce MERN Stack Application

A full-stack E-Commerce web application built using the MERN stack (MongoDB, Express.js, React, Node.js). This project demonstrates a modern, scalable architecture with authentication, product management, cart functionality, and order processing.

⸻

🚀 Features

👤 User Features
	•	User registration & login (JWT authentication)
	•	Browse products with categories
	•	Add/remove items from cart
	•	Checkout and place orders

🛠️ Admin Features
	•	Admin authentication & authorization
	•	Create, update, and delete products
	•	Manage product inventory
	•	View and manage customer orders

⸻

🧱 Tech Stack

Backend
	•	Node.js
	•	Express.js
	•	JWT for authentication
	•	Bcrypt for password hashing
	•	Mongoose for MongoDB ODM

Database
	•	MongoDB (NoSQL)


⸻

🔐 Authentication Flow
	1.	User logs in / registers
	2.	Server issues a JWT token
	3.	Token is stored on the client (HTTP-only cookie)
	4.	Protected routes validate JWT via middleware
